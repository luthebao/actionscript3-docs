# Package Detail

The flash.filesystem package contains classes used in accessing the filesystem. This package is only available to content running in the AIR runtime.

  

* * *

Classes

| Class| Description  
---|---|---  
| [File](File.html)|  A File object represents a path to a file or directory.  
| [FileMode](FileMode.html)|  The FileMode class defines string constants used in the `fileMode` parameter of the `open()` and `openAsync()` methods of the FileStream class.  
| [FileStream](FileStream.html)|  A FileStream object is used to read and write files.  
| [StorageVolume](StorageVolume.html)|  A StorageVolume object includes properties defining a mass storage volume.  
| [StorageVolumeInfo](StorageVolumeInfo.html)|  The StorageVolumeInfo object dispatches a StorageVolumeChangeEvent object when a storage volume is mounted or unmounted.
