# Igraphicsfill

Package| [flash.display](package-detail.html)  
---|---  
Interface| public interface IGraphicsFill  
Implementors| [GraphicsBitmapFill](../display/GraphicsBitmapFill.html), [GraphicsEndFill](../display/GraphicsEndFill.html), [GraphicsGradientFill](../display/GraphicsGradientFill.html), [GraphicsShaderFill](../display/GraphicsShaderFill.html), [GraphicsSolidFill](../display/GraphicsSolidFill.html)  
  
**Language version:**|  ActionScript 3.0   
---|---  
**Runtime version:**|  AIR 1.5   
---|---  
  
This interface is used to define objects that can be used as fill parameters in the flash.display.Graphics methods and drawing classes. Use the implementor classes of this interface to create and manage fill property data, and to reuse the same data for different instances.
