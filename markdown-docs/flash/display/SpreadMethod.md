The SpreadMethod class provides values for the `spreadMethod` parameter in the `beginGradientFill()` and `lineGradientStyle()` methods of the Graphics class. 

The following example shows the same gradient fill using various spread methods:

![linear gradient with SpreadMethod.PAD](../../images/beginGradientFill_spread_pad.jpg) | ![linear gradient with SpreadMethod.REFLECT](../../images/beginGradientFill_spread_reflect.jpg) | ![linear gradient with SpreadMethod.REPEAT](../../images/beginGradientFill_spread_repeat.jpg)  
---|---|---  
`SpreadMethod.PAD` | `SpreadMethod.REFLECT` | `SpreadMethod.REPEAT`
