# Package Detail

The flash.desktop package contains classes used for copy-and-paste and drag-and-drop operations, as well as the Icon class, used to define system icons used by a file.

  

* * *

Interfaces

| Interface| Description  
---|---|---  
|  _[IFilePromise](IFilePromise.html)_|  The IFilePromise interface defines the interface used by the AIR runtime to read data for a file promise.  
  
Classes

| Class| Description  
---|---|---  
| [Clipboard](Clipboard.html)|  The Clipboard class provides a container for transferring data and objects through the clipboard.  
| [ClipboardFormats](ClipboardFormats.html)|  The ClipboardFormats class defines constants for the names of the standard data formats used with the Clipboard class.  
| [ClipboardTransferMode](ClipboardTransferMode.html)|  The ClipboardTransferMode class defines constants for the modes used as values of the `transferMode` parameter of the `Clipboard.getData()` method.  
| [DockIcon](DockIcon.html)|  The DockIcon class represents the Mac OS X®-style dock icon.  
| [Icon](Icon.html)|  The Icon class represents an operating system icon.  
| [InteractiveIcon](InteractiveIcon.html)|  The InteractiveIcon class is the base class for the operating system icons associated with applications.  
| [InvokeEventReason](InvokeEventReason.html)|  The InvokeEventReason class enumerates values returned by the `reason` property of an `InvokeEvent` object.  
| [NativeApplication](NativeApplication.html)|  The NativeApplication class represents this AIR application.  
| [NativeDragActions](NativeDragActions.html)|  The NativeDragActions class defines string constants for the names of the drag-and-drop actions.  
| [NativeDragManager](NativeDragManager.html)|  The NativeDragManager class coordinates drag-and-drop operations.  
| [NativeDragOptions](NativeDragOptions.html)|  The NativeDragOptions class defines constants for the names of drag-and-drop actions allowed in a drag-and-drop operation.  
| [NativeProcess](NativeProcess.html)|  The NativeProcess class provides command line integration and general launching capabilities.  
| [NativeProcessStartupInfo](NativeProcessStartupInfo.html)|  This class provides the basic information used to start a process on the host operating system.  
| [NotificationType](NotificationType.html)|  The NotificationType class defines constants for use in the `priority` parameter of the DockIcon `bounce()` method and the `type` parameter of the NativeWindow `notifyUser()` method.  
| [SystemIdleMode](SystemIdleMode.html)|  The SystemIdleMode class provides constant values for system idle behaviors.  
| [SystemTrayIcon](SystemTrayIcon.html)|  The SystemTrayIcon class represents the Windows® taskbar notification area (system tray)-style icon.  
| [Updater](Updater.html)|  The Updater class is used to update the currently running application with a different version.
