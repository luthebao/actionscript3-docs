The Point object represents a location in a two-dimensional coordinate system, where _x_ represents the horizontal axis and _y_ represents the vertical axis. 

The following code creates a point at (0,0):
    
    
    var myPoint:Point = new Point();
    
    
    var myPoint = new Point();

Methods and properties of the following classes use Point objects:

  * BitmapData
  * DisplayObject
  * DisplayObjectContainer
  * DisplacementMapFilter
  * NativeWindow
  * Matrix
  * Rectangle

You can use the `new Point()` constructor to create a Point object.
