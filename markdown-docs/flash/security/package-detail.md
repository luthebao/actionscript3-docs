# Package Detail

This package includes classes for validating XML signatures. This package is only available to content running in the AIR runtime.

  

* * *

Interfaces

| Interface| Description  
---|---|---  
|  _[IURIDereferencer](IURIDereferencer.html)_|  IURIDereferencer defines an interface for objects that resolve URIs in an XML signature.  
  
Classes

| Class| Description  
---|---|---  
| [CertificateStatus](CertificateStatus.html)|  The CertificateStatus class defines constants used to report the results of certificate validation processing by a SecureSocket object.  
| [ReferencesValidationSetting](ReferencesValidationSetting.html)|  The ReferencesValidationSetting class defines constants used by the `referencesValidationSetting` property of an XMLSignatureValidator object.  
| [RevocationCheckSettings](RevocationCheckSettings.html)|  The RevocationCheckSettings class defines constants used by the revocationCheckSetting property of an XMLSignatureValidator object.  
| [SignatureStatus](SignatureStatus.html)|  The SignatureStatus class defines constants used by the validityStatus property of an XMLSignatureValidator object.  
| [SignerTrustSettings](SignerTrustSettings.html)|  The SignerTrustSettings class defines constants used with the signerTrustSettings property of an XMLSignatureValidator object.  
| [X500DistinguishedName](X500DistinguishedName.html)|  The X500DistinguishedName class defines Distinguished Name (DN) properties for use in the X509Certificate class.  
| [X509Certificate](X509Certificate.html)|  The X509Certificate class represents an X.509 certificate.  
| [XMLSignatureValidator](XMLSignatureValidator.html)|  The XMLSignatureValidator class validates whether an XML signature file is well formed, unmodified, and, optionally, whether it is signed using a key linked to a trusted digital certificate.
