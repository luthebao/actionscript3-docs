# Package Detail

The flash.filters package contains classes for bitmap filter effects. Filters let you apply rich visual effects, such as blur, bevel, glow, and drop shadows, to display objects.

  

* * *

Classes

| Class| Description  
---|---|---  
| [BevelFilter](BevelFilter.html)|  The BevelFilter class lets you add a bevel effect to display objects.  
| [BitmapFilter](BitmapFilter.html)|  The BitmapFilter class is the base class for all image filter effects.  
| [BitmapFilterQuality](BitmapFilterQuality.html)|  The BitmapFilterQuality class contains values to set the rendering quality of a BitmapFilter object.  
| [BitmapFilterType](BitmapFilterType.html)|  The BitmapFilterType class contains values to set the type of a BitmapFilter.  
| [BlurFilter](BlurFilter.html)|  The BlurFilter class lets you apply a blur visual effect to display objects.  
| [ColorMatrixFilter](ColorMatrixFilter.html)|  The ColorMatrixFilter class lets you apply a 4 x 5 matrix transformation on the RGBA color and alpha values of every pixel in the input image to produce a result with a new set of RGBA color and alpha values.  
| [ConvolutionFilter](ConvolutionFilter.html)|  The ConvolutionFilter class applies a matrix convolution filter effect.  
| [DisplacementMapFilter](DisplacementMapFilter.html)|  The DisplacementMapFilter class uses the pixel values from the specified BitmapData object (called the _displacement map image_) to perform a displacement of an object.  
| [DisplacementMapFilterMode](DisplacementMapFilterMode.html)|  The DisplacementMapFilterMode class provides values for the `mode` property of the DisplacementMapFilter class.  
| [DropShadowFilter](DropShadowFilter.html)|  The DropShadowFilter class lets you add a drop shadow to display objects.  
| [GlowFilter](GlowFilter.html)|  The GlowFilter class lets you apply a glow effect to display objects.  
| [GradientBevelFilter](GradientBevelFilter.html)|  The GradientBevelFilter class lets you apply a gradient bevel effect to display objects.  
| [GradientGlowFilter](GradientGlowFilter.html)|  The GradientGlowFilter class lets you apply a gradient glow effect to display objects.  
| [ShaderFilter](ShaderFilter.html)|  The ShaderFilter class applies a filter by executing a shader on the object being filtered.
