# Package Detail

The flash.accessibility package contains classes for supporting accessibility in Flash content and applications.

  

* * *

Interfaces

| Interface| Description  
---|---|---  
|  _[ISearchableText](ISearchableText.html)_|  The ISearchableText interface can be implemented by objects that contain text which should be searchable on the web.  
|  _[ISimpleTextSelection](ISimpleTextSelection.html)_|  The ISimpleTextSelection class can be used to add support for the MSAA ISimpleTextSelection interface to an AccessibilityImplementation.  
  
Classes

| Class| Description  
---|---|---  
| [Accessibility](Accessibility.html)|  The Accessibility class manages communication with screen readers.  
| [AccessibilityImplementation](AccessibilityImplementation.html)|  The AccessibilityImplementation class is the base class in Flash Player that allows for the implementation of accessibility in components.  
| [AccessibilityProperties](AccessibilityProperties.html)|  The AccessibilityProperties class lets you control the presentation of Flash objects to accessibility aids, such as screen readers.
