# Package Detail

The flash.display3D package contains the classes that the Flash Player uses to render geometrically defined graphics.

  

* * *

Classes

| Class| Description  
---|---|---  
| [Context3D](Context3D.html)|  The Context3D class provides a context for rendering geometrically defined graphics.  
| [Context3DBlendFactor](Context3DBlendFactor.html)|  Defines the values to use for specifying the source and destination blend factors.  
| [Context3DBufferUsage](Context3DBufferUsage.html)|  Defines the values to use for specifying the buffer usage type.  
| [Context3DClearMask](Context3DClearMask.html)|  Defines the values to use for specifying Context3D clear masks.  
| [Context3DCompareMode](Context3DCompareMode.html)|  Defines the values to use for specifying 3D buffer comparisons in the setDepthTest() and setStencilAction() methods of a Context3D instance.  
| [Context3DFillMode](Context3DFillMode.html)|  Defines the values to use for specifying 3D render fill mode in the setFillMode() methods of a Context3D instance.  
| [Context3DMipFilter](Context3DMipFilter.html)|  Defines the values to use for sampler mipmap filter mode   
| [Context3DProfile](Context3DProfile.html)|  Defines the values to use for specifying the Context3D profile.  
| [Context3DProgramType](Context3DProgramType.html)|  Defines the values to use for specifying whether a shader program is a fragment or a vertex program.  
| [Context3DRenderMode](Context3DRenderMode.html)|  Defines the values to use for specifying the Context3D render mode.  
| [Context3DStencilAction](Context3DStencilAction.html)|  Defines the values to use for specifying stencil actions.  
| [Context3DTextureFilter](Context3DTextureFilter.html)|  Defines the values to use for sampler filter mode.  
| [Context3DTextureFormat](Context3DTextureFormat.html)|  Defines the values to use for specifying a texture format.  
| [Context3DTriangleFace](Context3DTriangleFace.html)|  Constants to specify the orientation of a triangle relative to the view point.  
| [Context3DVertexBufferFormat](Context3DVertexBufferFormat.html)|  Defines the values to use for specifying vertex buffers.  
| [Context3DWrapMode](Context3DWrapMode.html)|  Defines the values to use for sampler wrap mode   
| [IndexBuffer3D](IndexBuffer3D.html)|  IndexBuffer3D is used to represent lists of vertex indices comprising graphic elements retained by the graphics subsystem.  
| [Program3D](Program3D.html)|  The Program3D class represents a pair of rendering programs (also called "shaders") uploaded to the rendering context.  
| [VertexBuffer3D](VertexBuffer3D.html)|  The VertexBuffer3D class represents a set of vertex data uploaded to a rendering context.
