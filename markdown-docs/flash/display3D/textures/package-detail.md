# Package Detail

The flash.display3D.textures package contains classes for textures.

  

* * *

Classes

| Class| Description  
---|---|---  
| [CubeTexture](CubeTexture.html)|  The CubeTexture class represents a cube texture uploaded to a rendering context.  
| [RectangleTexture](RectangleTexture.html)|  The Rectangle Texture class represents a 2-dimensional texture uploaded to a rendering context.  
| [Texture](Texture.html)|  The Texture class represents a 2-dimensional texture uploaded to a rendering context.  
| [TextureBase](TextureBase.html)|  The TextureBase class is the base class for Context3D texture objects.  
| [VideoTexture](VideoTexture.html)|  Prior to Flash Player 21, the use of video in Stage3D required the use of the Video object (which is not hardware accelerated), copying of video frame to a BitmapData object, and loading of the data onto the GPU which is CPU intensive.
