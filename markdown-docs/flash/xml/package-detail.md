# Package Detail

The flash.xml package contains Flash Player's legacy XML support, and other Flash Player-specific XML functionality.

  

* * *

Classes

| Class| Description  
---|---|---  
| [XMLDocument](XMLDocument.html)|  The XMLDocument class represents the legacy XML object that was present in ActionScript 2.0.  
| [XMLNode](XMLNode.html)|  The XMLNode class represents the legacy XML object that was present in ActionScript 2.0 and that was renamed in ActionScript 3.0.  
| [XMLNodeType](XMLNodeType.html)|  The XMLNodeType class contains constants used with `XMLNode.nodeType`.
