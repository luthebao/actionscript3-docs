# Package Detail

The flash.ui package contains user interface classes, such as classes for interacting with the mouse and keyboard.

  

* * *

Classes

| Class| Description  
---|---|---  
| [ContextMenu](ContextMenu.html)|  The ContextMenu class provides control over the items displayed in context menus.  
| [ContextMenuBuiltInItems](ContextMenuBuiltInItems.html)|  The ContextMenuBuiltInItems class describes the items that are built in to a context menu.  
| [ContextMenuClipboardItems](ContextMenuClipboardItems.html)|  The ContextMenuClipboardItems class lets you enable or disable the commands in the clipboard context menu.  
| [ContextMenuItem](ContextMenuItem.html)|  The ContextMenuItem class represents an item in the context menu.  
| [GameInput](GameInput.html)|  The `GameInput` class is the entry point into the GameInput API.  
| [GameInputControl](GameInputControl.html)|  The GameInputControl class represents a control on an input device.  
| [GameInputDevice](GameInputDevice.html)|  The `GameInputDevice` class represents a single input device commonly used for gaming.  
| [Keyboard](Keyboard.html)|  The Keyboard class is used to build an interface that can be controlled by a user with a standard keyboard.  
| [KeyboardType](KeyboardType.html)|  The KeyboardType class is an enumeration class that provides values for different categories of physical computer or device keyboards.  
| [KeyLocation](KeyLocation.html)|  The KeyLocation class contains constants that indicate the location of a key pressed on the keyboard or keyboard-like input device.  
| [Mouse](Mouse.html)|  The methods of the Mouse class are used to hide and show the mouse pointer, or to set the pointer to a specific style.  
| [MouseCursor](MouseCursor.html)|  The MouseCursor class is an enumeration of constant values used in setting the `cursor` property of the Mouse class.  
| [MouseCursorData](MouseCursorData.html)|  The MouseCursorData class lets you define the appearance of a "native" mouse cursor.  
| [Multitouch](Multitouch.html)|  The Multitouch class manages and provides information about the current environment's support for handling contact from user input devices, including contact that has two or more touch points (such as a user's fingers on a touch screen).
