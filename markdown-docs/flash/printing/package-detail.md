# Package Detail

The flash.printing package contains classes for printing Flash-based content.

  

* * *

Classes

| Class| Description  
---|---|---  
| [PaperSize](PaperSize.html)|  This class provides the available values for the `paperSize` parameter of the `PrintJob.selectPaperSize()` method.  
| [PrintJob](PrintJob.html)|  The PrintJob class lets you create content and print it to one or more pages.  
| [PrintJobOptions](PrintJobOptions.html)|  The PrintJobOptions class contains properties to use with the `options` parameter of the `PrintJob.addPage()` method.  
| [PrintJobOrientation](PrintJobOrientation.html)|  This class provides values that are used by the `PrintJob.orientation` property for the image position of a printed page.  
| [PrintMethod](PrintMethod.html)|  This class provides values for the `PrintJobOptions.printMethod` property to specify the method of printing a page.  
| [PrintUIOptions](PrintUIOptions.html)|  The PrintUIOptions class is used to specify options for print dialogs that are displayed to the user.
